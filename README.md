# React Lottery Game

## Overview
The **React Lottery Game** is a simple and interactive lottery game built using React and Vite. Players generate a random three-digit number, and if the sum of its digits equals **21** or **3**, they win the lottery!

## Features
- Randomly generates a three-digit number
- Calculates the sum of the digits automatically
- Displays winning or losing messages based on the sum
- Simple and user-friendly UI

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/abhi5heksah/Lottery.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Lottery
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## How to Play
1. Click the **Generate Number** button.
2. The game will display a random three-digit number.
3. The sum of the digits is calculated automatically.
4. If the sum equals **21** or **3**, you win!
5. Otherwise, try again.
